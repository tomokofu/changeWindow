●概要
　・本ツールは、Windows用のアプリケーションのサイズと位置を保存・設定するツールです。
　　毎日同じアプリケーションを開くのに、位置やサイズが毎回異なるのが嫌な方や、
　　大量のフォルダを開く方は使用してみてください。
　　ウインドウ群の表示サイズや位置を固定化させることで、作業効率が上がりことがあります。
　・フリーソフト(広告表示あり)です。
　・exeやbatファイルを実行するだけで動作します。インストール不要です。
　・フォルダ内のショートカットファイルを起動するツールも付属しています。
　　毎日起動するアプリケーションをショートカットとして配置すれば、一括で起動可能です。

●ファイルについて
　・実行ファイル
　　・set.bat
　　　サイズと位置を設定
　　・setMultiDisplay.bat
　　　サイズと位置を設定(デュアルディスプレイ用)
　　・save.bat
　　　サイズと位置を保存
　　・gui.bat
　　　GUIを表示
　　・all_active.bat
　　　起動中のウィンドウのアクティブ化
　　・all_minimized.bat
　　　起動中のウィンドウの最小化
　　・all_stop.bat
　　　起動中のウィンドウの停止
　　・changeWindow.exe
　　　実行ファイル本体
　　　動作モード(保存・設定)により、以下の引数を指定可能です。
　　　　第一引数：動作モード(保存：import、設定：export)を指定
　　　　　　　　　デフォルト：imoprt
　　　　第二引数：設定ファイル保存ディレクトリパス
　　　　　　　　　デフォルト：.\\
　　　　例
　　　　　保存
　　　　　　changeWindow.exe export .\\
　　　　　設定
　　　　　　changeWindow.exe import .\\
　　・commonConfig.csv
　　　共通設定ファイル
　　・programConfig.csv
　　　プログラム毎の設定ファイル
　　・0.run.bat
　　　ショットカット(リンク)ファイルを起動するツール
　　　以下の文字列から始まるファイルは起動しません。
　　　　no_run.
　　　　wait_start.
　　・0.waitRun.bat
　　　一定秒(10秒)待って、ショットカット(リンク)ファイルを起動するツール
　　　以下の文字列から始まるファイルが対象です。
　　　　wait_start.
　・explorer_windows.csv
　　サイズ・位置データ保存ファイル
　・ログファイル
　　・log\logfile.log
　　　自動生成されます

●使用方法の補足
　・保存方法
　　１．現在開いているウインドウの位置やサイズを「save.bat」を実行して保存ください。
　　　　記憶させたいウインドウを開き、好みの位置やサイズにしてから実行ください。
　　　　例：毎回開くフォルダ群やアプリケーション群等
　　２．現在開いているウインドウのショートカットを作成し、フォルダに配置ください。
　・設定方法
　　１．上記の保存方法の「２．」でショートカットを作成した場合、「0.run.bat」を実行ください。
　　　　ショートカットが順次実行され、アプリケーション等が起動します。
　　２．アプリケーションが起動してウインドウが表示されたら、「set.bat」を実行して、
　　　　位置やサイズを変更ください。
　・GUIでいろいろ操作
　　１．「gui.bat」を実行ください
　　　　保存された設定を表で表示します。「run」で起動し、「set」でウィンドウ幅や位置を変更可能です。
　　　　multi controlで、チェックしたら、画面下のボタンで一括操作も可能です。
